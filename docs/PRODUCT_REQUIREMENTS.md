# RXFG-RXSR Product Requirements

## Vision
A standalone Windows application providing real-time Super Resolution (RXSR) and Frame Generation (RXFG) for games.

## Core Features
- RXSR can run independently.
- RXFG can run independently.
- RXSR + RXFG can run together.
- Per-game profiles.
- Real-time monitoring.
- Overlay support.
- Diagnostics and logging.

## Target Platforms
- Windows 10
- Windows 11

## Supported APIs
- DirectX 11
- DirectX 12
- Vulkan

## Non-Goals (v1)
- Linux support
- Console support
- AI model training
- Cloud processing

## Success Criteria
- Stable operation
- Low latency overhead
- Easy setup
- Modern UI
- Broad hardware compatibility
